# TaskBoard - Gestor de Tareas Interactivo

## Descripción del proyecto
TaskBoard es una aplicación web desarrollada como proyecto escolar para la materia de programación.  
El objetivo del proyecto es aplicar conocimientos de React, HTML y JavaScript en la creación de una herramienta práctica que permita organizar tareas de manera visual e interactiva.

La aplicación se basa en un tablero tipo Kanban, donde las tareas pueden moverse entre columnas según su estado:
- Pendientes
- En progreso
- Completadas

Cada tarea se puede crear, editar, eliminar y mover con la funcionalidad de arrastrar y soltar.  
Los datos se guardan localmente en el navegador mediante LocalStorage.

## Funcionalidades principales
- Crear nuevas tareas con título, descripción y prioridad.  
- Editar o eliminar tareas existentes.  
- Mover tareas entre columnas.  
- Guardar automáticamente las tareas en LocalStorage.  
- Eliminar todas las tareas completadas.  
- Diseño adaptable a distintos tamaños de pantalla.

## Tecnologías utilizadas
- React  
- HTML5  
- JavaScript
